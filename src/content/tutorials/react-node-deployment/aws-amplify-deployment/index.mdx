# AWS Amplify Deployment: Complete React App Guide

AWS Amplify is a powerful platform for deploying full-stack applications. Learn how to deploy React apps with automatic CI/CD, custom domains, and production optimizations.

## What is AWS Amplify?

AWS Amplify is a development platform that provides a complete set of tools and services for building, deploying, and managing full-stack applications on AWS.

### Key Features

- **Hosting**: Static website hosting with global CDN
- **CI/CD**: Automatic deployments from Git repositories
- **Custom Domains**: Easy domain management
- **Environment Variables**: Secure configuration management
- **Backend Services**: Authentication, APIs, and databases
- **Monitoring**: Built-in analytics and monitoring

## Getting Started

### Prerequisites

- AWS Account
- Node.js 18+ installed
- Git repository with React app
- AWS CLI configured (optional)

### Install Amplify CLI

```bash
# Install Amplify CLI globally
npm install -g @aws-amplify/cli

# Configure Amplify CLI
amplify configure

# Verify installation
amplify --version
```

### Method 3: GitHub Actions Integration

```yaml
# .github/workflows/amplify-deploy.yml
name: Deploy to AWS Amplify

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
    - name: Checkout code
      uses: actions/checkout@v3
      
    - name: Setup Node.js
      uses: actions/setup-node@v3
      with:
        node-version: '18'
        cache: 'npm'
        
    - name: Install dependencies
      run: npm ci
      
    - name: Build application
      run: npm run build
      
    - name: Deploy to Amplify
      uses: aws-actions/amplify-cli-action@v1
      with:
        amplify-version: latest
        amplify-command: publish
        amplify-app-id: ${{ secrets.AMPLIFY_APP_ID }}
        amplify-branch: main
        amplify-env: production
      env:
        AWS_ACCESS_KEY_ID: ${{ secrets.AWS_ACCESS_KEY_ID }}
        AWS_SECRET_ACCESS_KEY: ${{ secrets.AWS_SECRET_ACCESS_KEY }}
        AWS_REGION: us-east-1
```

## Custom Domains

### Adding Custom Domain

1. **In Amplify Console**
   - Go to Domain management
   - Click "Add domain"
   - Enter your domain name
   - Follow DNS configuration instructions

2. **DNS Configuration**
   ```dns
   # Add CNAME record
   www.yourdomain.com CNAME d1234567890.cloudfront.net
   
   # Add A record for root domain
   yourdomain.com A 192.0.2.1
   ```

3. **SSL Certificate**
   - Amplify automatically provisions SSL certificates
   - Supports both www and non-www domains
   - Automatic certificate renewal

### Domain Verification

```bash
# Verify domain ownership
# Add TXT record to your DNS:
# _amazonses.yourdomain.com TXT "verification-token"
```

### Using Authentication in React

```javascript
// src/components/Auth.js
import { Auth } from 'aws-amplify';
import { useState, useEffect } from 'react';

function AuthComponent() {
  const [user, setUser] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    checkAuthState();
  }, []);

  const checkAuthState = async () => {
    try {
      const currentUser = await Auth.currentAuthenticatedUser();
      setUser(currentUser);
    } catch (error) {
      setUser(null);
    } finally {
      setLoading(false);
    }
  };

  const signIn = async (username, password) => {
    try {
      const user = await Auth.signIn(username, password);
      setUser(user);
    } catch (error) {
      console.error('Sign in error:', error);
    }
  };

  const signOut = async () => {
    try {
      await Auth.signOut();
      setUser(null);
    } catch (error) {
      console.error('Sign out error:', error);
    }
  };

  if (loading) return <div>Loading...</div>;

  return (
    <div>
      {user ? (
        <div>
          <p>Welcome, {user.username}!</p>
          <button onClick={signOut}>Sign Out</button>
        </div>
      ) : (
        <div>
          <button onClick={() => signIn('username', 'password')}>
            Sign In
          </button>
        </div>
      )}
    </div>
  );
}

export default AuthComponent;
```

### Using API in React

```javascript
// src/services/api.js
import { API } from 'aws-amplify';

const apiName = 'myapi';
const path = '/items';

export const apiService = {
  async getItems() {
    try {
      const response = await API.get(apiName, path);
      return response;
    } catch (error) {
      console.error('Error fetching items:', error);
      throw error;
    }
  },

  async createItem(item) {
    try {
      const response = await API.post(apiName, path, { body: item });
      return response;
    } catch (error) {
      console.error('Error creating item:', error);
      throw error;
    }
  },

  async updateItem(id, item) {
    try {
      const response = await API.put(apiName, `${path}/${id}`, { body: item });
      return response;
    } catch (error) {
      console.error('Error updating item:', error);
      throw error;
    }
  },

  async deleteItem(id) {
    try {
      const response = await API.del(apiName, `${path}/${id}`);
      return response;
    } catch (error) {
      console.error('Error deleting item:', error);
      throw error;
    }
  }
};
```

### Using Analytics in React

```javascript
// src/App.js
import { Analytics } from 'aws-amplify';
import { useEffect } from 'react';

function App() {
  useEffect(() => {
    // Record page view
    Analytics.record({ name: 'pageView' });
  }, []);

  const handleButtonClick = () => {
    // Record custom event
    Analytics.record({
      name: 'buttonClick',
      attributes: {
        buttonName: 'submit',
        page: 'home'
      }
    });
  };

  return (
    <div className="App">
      <button onClick={handleButtonClick}>
        Submit
      </button>
    </div>
  );
}
```
